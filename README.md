### Реализованно:
- Создан User с полями id, email, password, age
- К полям применены аннотации @Min @Max @NotBlank, а так же кастомные валидаторы:

1) EmailValidator - для адреса электоронной почты. Почта подчиняется паттерну и другим условиям.
2) PasswordValidator - для пароля, содержит несколько других встроенных валидаторов.

Так же интернационализация отражена в папке resources.
