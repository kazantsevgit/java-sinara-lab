# Java Spring Boot
Учебный проект для выполнения заданий по Spring Boot через Pull Requests.

## Описание

Репозиторий создан для прохождения лабораторных работ по Spring Boot. Основной фокус - освоение workflow совместной разработки через систему Pull Requests.

## Цели проекта

- Освоение Spring Boot на практике
- Изучение Git workflow через Pull Requests
- Практика code review процессов
- Работа в командной разработке

## Технологии

- Java
- Spring Boot
- Maven/Gradle
- Git & GitHub

## Процесс работы

Выполнение заданий осуществляется через:
1. Создание feature-веток
2. Реализацию задач
3. Открытие Pull Request
4. Code review
5. Мердж в основную ветку
